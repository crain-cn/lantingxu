兰亭序续写 API（MCP）

接口通过 MCP（Model Context Protocol） 暴露，JSON-RPC 2.0，端点：POST /api/mcp。

方法

  • initialize — 协商协议与能力
  • tools/list — 列出可用 tools（获取 JWT、随机故事、续写、打分、知乎发布等）
  • tools/call — 调用指定 tool（name + arguments）

示例：tools/list

{"jsonrpc":"2.0","id":1,"method":"tools/list"}

示例：tools/call（获取 JWT）

{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"get_jwt_token","arguments":{"appId":"default","appSecret":"你的密钥"}}}

需认证的 tool 可在 arguments 中传 accessToken，或在请求头带 Authorization: Bearer <token>（SecondMe OpenClaw 由平台转发 Bearer，与兰亭 JWT 二选一即可）。

工具 secondme_user_info：用当前 Bearer 调 SecondMe 用户信息，返回含 data.userId；需应用 scope user.info。

OpenClaw 集成说明见仓库 docs/SECONDME_OPENCLAW.md。

REST 兼容：原有 /api/openapi/* 路径仍可用（JWT 认证方式不变）。
